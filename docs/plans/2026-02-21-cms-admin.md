# CMS Admin Dashboard Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a CMS admin interface for restaurant owners to view performance stats, configure UI styles, and manage modular views (smart suggestions, group ordering, cross-sales).

**Architecture:** We will create a new group in Next.js App Router for `/admin`. It will include `/admin/dashboard`, `/admin/appearance`, and `/admin/modules`. We will use existing Tailwind CSS and components. State will be managed locally for previewing before "publishing" to the backend.

**Tech Stack:** Next.js (App Router), React, TailwindCSS, Lucide React (icons).

---
